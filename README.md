# algorithm

其中CUDA的项目是在VS2013和CUDA toolkit7.5的环境上编写的，参考了CUDA原有的samples。

1.CUDA简单例程：
实现了一个较为简单的GPU编程例子，方便入门。

2.算法设计-智力拼图：
其中包括了单线程和多线程的程序，GPU程序。
单线程和多线程的程序只有一个cpp文件，兼容了windows和linux系统，均需要使用c++11，其中linux上面遍历需要加上-lpthread。
GPU的程序是使用了层序遍历实现智力拼图问题的求解。
